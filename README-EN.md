## Overview

**[013] Variational-Backpack** is AlphaLabs' dual-platform trading strategy plugin, supporting perpetual futures trading on both Lighter and Backpack exchanges.

### Key Features

- **Dual Platform Integration**: Supports both Variational DEX and Backpack CEX
- **Smart Order Management**: Dynamic order sizing and timing
- **Risk Control**: Volatility-based position management
- **Low Slippage Execution**: Optimized order execution strategy

---

## Getting Started

### Prerequisites

1. **AlphaLabs Signer Manager**: Installed and running
2. **Chrome Browser**: Version 109+
3. **AlphaLabs Account**: Register at [alphalabs.app](https://alphalabs.app)
4. **API Credentials**: Configure Variational and Backpack API keys

### Installation

1. Download and extract the plugin package
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the plugin directory
5. Configure API keys and login

## Usage

### Starting the Strategy

1. Ensure sufficient balance in Variational and Backpack accounts
2. Adjust configuration parameters
3. Click "Start Strategy"

### Stopping the Strategy

1. Click "Stop Strategy"
2. Wait for positions to close
